## Features

- **Random Quotes:** Refresh to get a new quote every time.  
- **Tweet Quotes:** Share your favorite quotes on Twitter with a single click.  
- **Beautiful UI:** Clean, minimalistic, and responsive design.  
- **Offline Ready:** Uses a local array of public-domain quotes, so it works even without internet.

## Installation & Usage

1. Clone the repository or download the ZIP.
2. Open `index.html` in your browser.
3. Click **New Quote** to get a random quote.
4. Click **Tweet** to share the quote on Twitter.

```bash
# Example:
git clone https://github.com/anjali012005/DynamicQuote-Generator.git
cd DynamicQuote-Generator
open index.html
```
Customization
You can add more quotes to quotes array in script.js for a larger variety.

Update styling in style.css to match your theme or brand.

Demo
A live demo can be hosted on GitHub Pages, Vercel, or Netlify for easy sharing.
